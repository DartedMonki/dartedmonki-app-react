## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/DartedMonki/dartedmonki-app-react.git
   ```
2. Install Yarn packages
   ```sh
   yarn install
   ```
4. Run in development
   ```sh
   yarn dev
   ```
   <p align="center">
     or you can do
   </p>
   
   ```sh
   yarn start
   ```
